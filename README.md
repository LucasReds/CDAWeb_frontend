# CDAWeb_frontend

### Como correr la app

- `cd tank-wars`
- `npm run dev`

### Como agregar rutas

- En archivo `Routing.jsx` agregar un nuevo `<Route path={ruta}>`

### TESTEAR LA API:

- Ir a `API TEST` en la navbar.
- Correr el repo de backend en local (recordar iniciar postgresql).
- Elegir un endpoint a testear y llenar los datos.
